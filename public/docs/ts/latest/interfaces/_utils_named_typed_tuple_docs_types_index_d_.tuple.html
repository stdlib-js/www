subtuple

  • subtuple(begin?: undefined | number, end?: undefined | number): Tuple
    • Defined in lib/node_modules/@stdlib/utils/named-typed-tuple/docs/types/index.d.ts:1217

    Creates a new named typed tuple over the same underlying ArrayBuffer and with the same underlying data type as the host tuple.

    Parameters

    • Optional begin: undefined | number

      start element index (inclusive); if less than zero, the start index is resolved relative to the last tuple element (default: 0)

    • Optional end: undefined | number

      end element index (exclusive); if less than zero, the end index is resolved relative to the last tuple element (default: tuple.length)

    Returns Tuple

    a new named typed tuple

    Example

    var factory = namedtypedtuple( [ 'x', 'y', 'z' ] );

var p1 = factory( [ 1.0, 0.0, -1.0 ] );

var p2 = p1.subtuple();

var bool = ( p1 === p2 );
// returns false

bool = ( p1.buffer === p2.buffer );
// returns true

var len = p2.length;
// returns 3

var x = p2.x;
// returns 1.0

var y = p2.y;
// returns 0.0

var z = p2.z;
// returns -1.0
