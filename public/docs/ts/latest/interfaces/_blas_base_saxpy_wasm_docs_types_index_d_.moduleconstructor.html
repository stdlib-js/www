Interface ModuleConstructor

Interface defining a module constructor which is both "newable" and "callable".

  • __call(mem: Memory): Module
    • Defined in lib/node_modules/@stdlib/blas/base/saxpy-wasm/docs/types/index.d.ts:81

    Returns a new WebAssembly module wrapper instance which uses the provided WebAssembly memory instance as its underlying memory.

    Parameters

    • mem: Memory

      WebAssembly memory instance

    Returns Module

    module wrapper instance

    Example

    var Memory = require( '@stdlib/wasm/memory' );
var oneTo = require( '@stdlib/array/one-to' );
var ones = require( '@stdlib/array/ones' );
var zeros = require( '@stdlib/array/zeros' );
var bytesPerElement = require( '@stdlib/ndarray/base/bytes-per-element' );

// Create a new memory instance with an initial size of 10 pages (640KiB) and a maximum size of 100 pages (6.4MiB):
var mem = new Memory({
    'initial': 10,
    'maximum': 100
});

// Create a BLAS routine:
var mod = saxpy.Module( mem );
// returns <Module>

// Initialize the routine:
mod.initializeSync();

// Define a vector data type:
var dtype = 'float32';

// Specify a vector length:
var N = 5;

// Define pointers (i.e., byte offsets) for storing two vectors:
var xptr = 0;
var yptr = N * bytesPerElement( dtype );

// Write vector values to module memory:
mod.write( xptr, oneTo( N, dtype ) );
mod.write( yptr, ones( N, dtype ) );

// Perform computation:
var ptr = mod.main( N, 5.0, xptr, 1, yptr, 1 );
// returns <number>

var bool = ( ptr === yptr );
// returns true

// Read out the results:
var view = zeros( N, dtype );
mod.read( yptr, view );
// view => <Float32Array>[ 6.0, 11.0, 16.0, 21.0, 26.0 ]
