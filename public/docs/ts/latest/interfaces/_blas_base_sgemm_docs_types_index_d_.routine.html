    • Defined in lib/node_modules/@stdlib/blas/base/sgemm/docs/types/index.d.ts:28

    Performs the matrix-matrix operation C = α*op(A)*op(B) + β*C where op(X) is either op(X) = X or op(X) = X^T, α and β are scalars, A, B, and C are matrices, with op(A) an M by K matrix, op(B) a K by N matrix, and C an M by N matrix.

    Parameters

    • order: Layout

      storage layout

    • transA: TransposeOperation

      specifies whether A should be transposed, conjugate-transposed, or not transposed

    • transB: TransposeOperation

      specifies whether B should be transposed, conjugate-transposed, or not transposed

    • M: number

      number of rows in the matrix op(A) and in the matrix C

    • N: number

      number of columns in the matrix op(B) and in the matrix C

    • K: number

      number of columns in the matrix op(A) and number of rows in the matrix op(B)

    • alpha: number

      scalar constant

    • A: Float32Array

      first matrix

    • LDA: number

      stride of the first dimension of A (a.k.a., leading dimension of the matrix A)

    • B: Float32Array

      second matrix

    • LDB: number

      stride of the first dimension of B (a.k.a., leading dimension of the matrix B)

    • beta: number

      scalar constant

    • C: Float32Array

      third matrix

    • LDC: number

      stride of the first dimension of C (a.k.a., leading dimension of the matrix C)

    Returns Float32Array

    C

    Example

    var Float32Array = require( '@stdlib/array/float32' );

var A = new Float32Array( [ 1.0, 2.0, 3.0, 4.0 ] );
var B = new Float32Array( [ 1.0, 1.0, 0.0, 1.0 ] );
var C = new Float32Array( [ 1.0, 2.0, 3.0, 4.0 ] );

sgemm( 'row-major', 'no-transpose', 'no-transpose', 2, 2, 2, 1.0, A, 2, B, 2, 1.0, C, 2 );
// C => <Float32Array>[ 2.0, 5.0, 6.0, 11.0 ]

    Performs the matrix-matrix operation C = α*op(A)*op(B) + β*C, using alternative indexing semantics and where op(X) is either op(X) = X or op(X) = X^T, α and β are scalars, A, B, and C are matrices, with op(A) an M by K matrix, op(B) a K by N matrix, and C an M by N matrix.

    Parameters

    • transA: TransposeOperation

      specifies whether A should be transposed, conjugate-transposed, or not transposed

    • transB: TransposeOperation

      specifies whether B should be transposed, conjugate-transposed, or not transposed

    • M: number

      number of rows in the matrix op(A) and in the matrix C

    • N: number

      number of columns in the matrix op(B) and in the matrix C

    • K: number

      number of columns in the matrix op(A) and number of rows in the matrix op(B)

    • alpha: number

      scalar constant

      first matrix

    Example

    var Float32Array = require( '@stdlib/array/float32' );

var A = new Float32Array( [ 1.0, 2.0, 3.0, 4.0 ] );
var B = new Float32Array( [ 1.0, 1.0, 0.0, 1.0 ] );
var C = new Float32Array( [ 1.0, 2.0, 3.0, 4.0 ] );

sgemm.ndarray( 'no-transpose', 'no-transpose', 2, 2, 2, 1.0, A, 2, 1, 0, B, 2, 1, 0, 1.0, C, 2, 1, 0 );
