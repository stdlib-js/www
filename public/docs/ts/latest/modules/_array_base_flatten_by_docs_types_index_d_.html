flattenBy: FlattenBy
  • Defined in lib/node_modules/@stdlib/array/base/flatten-by/docs/types/index.d.ts:1122

Flattens an n-dimensional nested array according to a callback function.

Notes
  • The function assumes that all nested arrays have the same length (i.e., the input array is not a ragged array).
param

input array

param

array shape

param

specifies whether to flatten array values in colexicographic order

param
