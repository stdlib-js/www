    • Defined in lib/node_modules/@stdlib/ndarray/base/from-scalar/docs/types/index.d.ts:109

    Returns a zero-dimensional ndarray containing a provided scalar value.

    Notes
    • If provided a number, the function returns a zero-dimensional ndarray containing a complex number whose real component equals the provided scalar value and whose imaginary component is zero.

    Parameters

    • value: number | ComplexLike

      scalar value

    • dtype: "complex128"

      array data type

    • order: Order

      memory layout (row-major or column-major)

    Returns complex128ndarray

    zero-dimensional ndarray

    Example

    var Complex128 = require( '@stdlib/complex/float64/ctor' );
var real = require( '@stdlib/complex/float64/real' );
var imag = require( '@stdlib/complex/float64/imag' );

var v = new Complex128( 1.0, 2.0 );

var x = scalar2ndarray( v, 'complex128', 'row-major' );
// returns <ndarray>

var sh = x.shape;
// returns []

var dt = x.dtype;
// returns 'complex128'

var v = x.get();
// returns <Complex128>

var re = real( v );
// returns 1.0

var im = imag( v );
// returns 2.0
