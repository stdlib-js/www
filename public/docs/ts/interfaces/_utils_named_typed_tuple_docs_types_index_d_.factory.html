    Creates a named typed tuple from a typed array, array-like object, or iterable.

    Parameters

    • obj: TypedArray | ArrayLike<number> | Iterable<number>

      typed array, array-like object, or iterable from which to generate a named typed tuple

    • Optional dtype: DType

      tuple data type

    Returns Tuple

    named typed tuple

    Example

    var factory = namedtypedtuple( [ 'x', 'y' ] );

var tuple = factory( [ 1.0, -1.0 ] );

var x = tuple.x;
// returns 1.0

x = tuple[ 0 ];
// returns 1.0

var y = tuple.y;
// returns -1.0

y = tuple[ 1 ];
// returns -1.0

    Example

    var Float64Array = require( `@stdlib/array/float64` );

var factory = namedtypedtuple( [ 'x', 'y' ] );

var tuple = factory( new Float64Array( [ 1.0, -1.0 ] ) );

var x = tuple.x;
// returns 1.0

x = tuple[ 0 ];
// returns 1.0

var y = tuple.y;
// returns -1.0

y = tuple[ 1 ];
// returns -1.0
