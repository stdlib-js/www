  • Overrides Readable.__constructor

    • Defined in lib/node_modules/@stdlib/random/streams/hypergeometric/docs/types/index.d.ts:85

    Returns a readable stream for generating a stream of pseudorandom numbers drawn from a hypergeometric distribution.

    throws

    N must be a nonnegative integer

    throws

    K must be a nonnegative integer

    throws

    n must be a nonnegative integer

    throws

    n must be less than or equal to N

    throws

    K must be less than or equal to N

    throws

    must provide valid options

    throws

    must provide a valid state

    Parameters

    • N: number

      population size

    • K: number

      subpopulation size

    • n: number

      number of draws

    • Optional options: Options

      stream options

    Returns RandomStream

    stream instance

    Example

    var inspectStream = require( `@stdlib/streams/node/inspect-sink` );

function log( chunk ) {
   console.log( chunk.toString() );
}

var opts = {
    'iter': 10
};

var stream = new RandomStream( 20, 10, 7, opts );

stream.pipe( inspectStream( log )  );
